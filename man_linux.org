* comm 对比两个文件的不同
  1. comm file1 file2
     | column | description                     |
     |--------+---------------------------------|
     |      1 | lines unique in file1           |
     |      2 | lines unique in file2           |
     |      3 | lines that appear in both files |
  2. columen [param] file1 file2
     |    param | description                                          |
     |----------+------------------------------------------------------|
     |       -1 | suppress column 1                                    |
     |       -2 | suppress column 2                                    |
     |       -3 | suppress column 3                                    |
     | 多个参数 | 去除对应column（eg. -12  suppress column 1 column 2) |
